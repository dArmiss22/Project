# Expense Tracker (Трекер расходов)

## Информация об авторе
**Имя и фамилия:** Скарга Дарья

## Описание программы

GUI-приложение для отслеживания личных расходов. Позволяет добавлять расходы с указанием суммы, категории и даты. Есть фильтрация по категории и дате, подсчёт суммы за выбранный период. Данные сохраняются в JSON-файл.

## Как использовать

1. Запустите файл `expense_tracker.py`.
2. Введите сумму, выберите категорию, введите дату в формате ДД.ММ.ГГГГ.
3. Нажмите «Добавить расход» — запись появится в таблице.
4. Для фильтрации выберите категорию или введите дату, нажмите «Применить».
5. Для подсчёта суммы за период введите даты «С» и «По», нажмите «Посчитать».
6. Все данные автоматически сохраняются в `expenses.json`.

## Формат даты

ДД.ММ.ГГГГ, например: 15.04.2026

## Тестовый сценарий

1. Добавьте расход: 500 руб., «Еда», 01.04.2026.
2. Добавьте расход: 200 руб., «Транспорт», 02.04.2026.
3. Добавьте расход: 1000 руб., «Развлечения», 05.04.2026.
4. Отфильтруйте по категории «Еда» — останется одна запись.
5. Сбросьте фильтры — все три записи снова видны.
6. Посчитайте сумму за период с 01.04.2026 по 03.04.2026 — должно быть 700 руб.
